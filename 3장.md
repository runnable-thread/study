# Docker 명령어 모음


## Docker search

- 이미지 검색기능 
- Docker hub 에서 이미지 공유 생태계가 구성되어있음


## Docker pull

- `docker pull <이미지 이름>:<태그>`


## Docker image 

- `sudo docker images`
 - 내가 가진 이미지 목록 출력 
## Docker ps

- `docker ps -a`
  - 모든 컨테이너 목록 출력
  
## Docker start
- 정지된 컨테이너 실행 

## Docker attach
- 접속
- exit 혹은 cntrl + c 로 나오면 컨테이너가 stop되지만, cntr + p , q 를 입력하면 정지하지 않고 백그라운드로 돌면서 나와짐

## Docker exec 
- 외부에서 컨테이너안의 명령
